
******************
Справочные системы
******************

В период работы 1996-1996 руководителем группы программистов в издательском доме "Золотые страницы" сформировал команду и разработал программы для работы ряда справочных систем.

Учет носителей медиа информации 
===============================

Заказчик Мэрия г.Москва, комитет по телекоммуникациям и сми.

Система учета изготовления, лицензирования и наклейки защитных знаков на носители информации.

Используемые технологии: MS SQL Server 7.0, VB 6.0, ASP, RAS

Интернет справочник Золотые страницы
====================================

Электронная версия телефонного справочника "Золотые страницы".

Используемые технологии: MS SQL Server 6.5, MS Visual Basic 5.0, MS ASP, MS IIS 4.0

Бэк-офис справочника Золотые страницы
=====================================

Система автоматизации деятельности отдела маркетинга и продаж.

Используемые технологии: MS SQL Server 6.5, MS Visual Basic 6.0, Delphi 2.0, 1C:Предприятие
